Produced by @whjab, inspired by **Dirsearch** by [@maurosoria](https://github.com/maurosoria)

*Send me a message on twitter: [@_whjab](https://x.com/_whjab)*

How to use
---------------
```
$ python tests/test_brute_forcer.py -u "URL" -w "WORDLIST"
```
